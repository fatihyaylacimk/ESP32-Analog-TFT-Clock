# ESP32 Analog TFT Clock

This project implements a real-time analog clock using ESP32, MicroPython, and an ILI9341 TFT display.

The system automatically connects to Wi-Fi, synchronizes the current time using NTP servers, and renders a fully graphical analog clock with hour, minute, and second hands.

## Features

- Wi-Fi connectivity
- NTP time synchronization
- Real-time analog clock
- ILI9341 TFT display support
- Dynamic graphical rendering
- Automatic timezone adjustment
- Smooth second-hand movement
- MicroPython implementation

## Hardware

- ESP32 Development Board
- ILI9341 TFT Display
